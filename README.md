# ADAS + LLM Capstone Project

## Overview
This repository showcases a professional capstone project combining:
- ADAS AI system architecture
- Automotive safety (ISO 26262, SOTIF)
- Large Language Models (LLMs) applied responsibly in automotive
- AI governance and deployment considerations

## Goals
- Demonstrate system-level understanding of ADAS AI
- Explore limitations and risks of AI in safety-critical systems
- Design a controlled LLM-based internal copilot for automotive teams

## Target Audience
- AI Technical Program Managers
- ADAS System Architects
- Automotive AI Leaders
